# HỆ THỐNG QUẢN LÝ CÔNG VIỆC

## Giới thiệu
Dự án Java quản lý công việc (Task Management System).

## Chức năng
- Đăng ký / Đăng nhập
- Thêm công việc
- Xem danh sách công việc
- Cập nhật trạng thái công việc
- Xóa công việc

## Công nghệ sử dụng
- Java
- OOP
- Scanner (Console UI)

## Cách chạy dự án
1. Clone project về máy:
   git clone <link-repo>

2. Mở bằng Eclipse

3. Chạy file Main.java
